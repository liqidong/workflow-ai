# 示例：事件驱动学习沉淀

## 请求

一次修复后发现：同类任务反复遗漏同一个验证步骤。

## Route

```text
Route: blocker
Spec layer: inherited
Spec control surface: <active-change>
Gate: n/a
Learning capture: triggered
Problem kind: repeated drift
Risk level: medium
Control surface: failing check + existing project learning log
Reason: repeated miss produced a durable workflow lesson
Verification required: corrected behavior + learning record location
```

## 示例说明

这个示例展示的是“学习沉淀只在事件触发时记录”。它不是每次任务都要执行的 gate，也不是新的 source of truth。
